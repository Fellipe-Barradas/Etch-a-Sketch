# Etch-a-Sketch
Etch a Sketch é um webapp para desenhar pixel-art, que permite escolher as dimensões da tela, entre outros artificios.

# Tecnologias
<ul>
<li>HTML</li>
<li>CSS</li>
<li>JAVASCRIPT</li>
</ul>
